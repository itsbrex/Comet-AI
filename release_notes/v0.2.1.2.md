# ☄️ Comet AI — v0.2.1.2 "The Secure DOM Update"
> *Enhanced security, screenshot embedding in PDFs, and expanded capability demonstrations.*

---

## 🚀 Key Highlights

### 🔒 Secure DOM Reading with Injection Detection
Comet AI now features **Secure DOM Reading** — a read-only mode with comprehensive security:

- **PII Redaction**: Automatically removes emails, phones, tokens, and credentials from page content
- **Injection Detection**: Blocks prompt injection patterns using the existing threat detection system
- **Filtered Element Tree**: Scripts, styles, navigation, ads, and tracking elements are automatically blocked
- **DOM Search**: New `[DOM_SEARCH: query]` command to search within the current page
- **DOM Read Filtered**: New `[DOM_READ_FILTERED: query]` command with comprehensive filtering

### 📸 Screenshot Embedding in PDFs
The PDF generator now supports **embedded screenshots and images**:

- **Automatic Capture**: AI can take screenshots during demonstrations
- **Image Support**: PNG/JPEG images can be embedded in PDF reports
- **Caption Support**: Images include optional captions
- **Responsive Sizing**: Images scale appropriately within the document

### 🎯 Enhanced [EXPLAIN_CAPABILITIES]
The capability demonstration now executes **real tasks live**:

1. **Web Search** — Fetches latest technology news in real-time
2. **Shell Commands** — Retrieves WiFi/network information based on OS
3. **System Control** — Adjusts volume to 50%
4. **App Launching** — Opens Calculator (macOS/Windows/Linux auto-detected)
5. **Screenshot Capture** — Takes and embeds screen images
6. **PDF Generation** — Creates branded reports with embedded screenshots
