| Feature | Description |
|---------|-------------|

| **Auto-save** | Last used model remembered |

---

## CLI Features

### Model Selection
- Reads configured models from browser `config.json`
- Shows all available models with numbers
- Interactive selection by typing number
- Last used model saved automatically

**Supported Providers:**
- ollama (local)
- openai (GPT)
- anthropic (Claude)
- gemini (Google)
- xai (Grok)
- groq

### Usage

```bash
# Ask with last used model
comet ask "Hello"

# Ask with specific model
comet ask "Hello" -m openai

# Show available models
comet models

# Interactive model selection
comet select-model

# Interactive chat mode
comet chat
```
