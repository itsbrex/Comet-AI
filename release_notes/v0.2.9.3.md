## 🍎 macOS Integration

### Siri & Apple Shortcuts

**URL Scheme (`comet-ai://`)**

All Comet actions accessible via Shortcuts app:

| URL | Action |
|-----|--------|
| `comet-ai://chat?message=...` | Send message to AI |
| `comet-ai://search?query=...` | Web search |
| `comet-ai://create-pdf?content=...&title=...` | Generate PDF |
| `comet-ai://navigate?url=...` | Open URL |
| `comet-ai://run-command?command=...` | Run command |
| `comet-ai://volume?level=0-100` | Set volume |
| `comet-ai://open-app?appName=...` | Open app |
| `comet-ai://screenshot` | Capture screen |
| `comet-ai::schedule?task=...&cron=...` | Schedule task |
| `comet-ai::ask-ai?prompt=...&speak=true` | Ask + Speak |

**Siri Voice Commands:**
- "Ask Comet AI [question]"
- "Search [query] with Comet AI"
- "Create PDF in Comet AI"
- "Run [command] in Comet AI"
- "Schedule [task] in Comet AI"

### Voice Control

- **Dictation**: Press ⌘⇧D to dictate
- **Text-to-Speech**: AI responses spoken aloud
- **AppleScript Bridge**: Pre-built automation scripts

### Raycast Integration

Comet supports Raycast extensions via URL scheme:
- Open Raycast → type "Comet" → use commands

| Feature | Description |

| Voice Control | PowerShell Speech Recognition + TTS |

### API Reference

```javascript
// Text-to-Speech
window.electronAPI.windows.voice.speak('Hello from Comet!');

// Speech Recognition  
window.electronAPI.windows.voice.listen({ timeout: 10000 });

// Get available voices
const voices = window.electronAPI.windows.voice.getVoices();

// Open Microsoft Copilot
window.electronAPI.windows.openCopilot();

// Create shortcut
window.electronAPI.windows.createShortcut('Chat AI', 'chat', { message: 'Hello' });
```

## 🔄 Native Click Alternatives

Cross-platform automation priority chain for reliable clicking:

| Priority | macOS | Windows | Linux |
|----------|-------|---------|-------|
| **1st** | steve CLI (AXUIElement) | nut.js | xdotool |
| **2nd** | AppleScript | xa11y | xte/xinput |
| **3rd** | robotjs | robotjs | robotjs |

### Benefits

- **steve CLI**: Native Accessibility API, works with all apps
- **nut.js**: Cross-platform, actively maintained
- **xa11y**: Cross-platform accessibility abstraction
- **Always falls back to robotjs** for compatibility

---

## 📷 Native OCR Alternatives

Cross-platform OCR for screen text recognition:

| Priority | macOS | Windows | Linux |
|----------|-------|---------|-------|
| **1st** | uniOCR (Vision) | uniOCR (Windows) | uniOCR (Tesseract) |
| **2nd** | RustO! (PaddleOCR) | RustO! | RustO! |
| **3rd** | Native Vision API | Windows OCR | AT-SPI2 |
| **4th** | Tesseract.js | Tesseract.js | Tesseract.js |

### Performance

- **uniOCR**: 3.2 images/second (macOS Vision), 90% accuracy
- **RustO!**: 99.3% accuracy, based on PaddleOCR
- **Tesseract.js**: 99%+ accuracy on printed text
