# Comet AI v0.2.9.4.1 Release Notes

**Release Date:** April 23, 2026  
**Version:** 0.2.9.4.1  
**Channel:** Stable  
**Codename:** Nebula  

---

## Highlights

This release introduces enhanced WiFi Sync with desktop control from mobile, improved QR code scanner, and better AI streaming support for seamless mobile-desktop integration.

### What's New

| Feature | Platform | Description |
|---------|----------|-------------|
| **Desktop Control** | Mobile | Control desktop AI chat from mobile app |
| **Enhanced QR Scanner** | Mobile | Improved QR code scanning for pairing |
| **AI Streaming** | Mobile | Real-time AI response streaming |
| **Sync Settings UI** | Mobile | Connection status and settings management |

---

## Mobile Features

### WiFi Sync Desktop Control

Control your desktop Comet AI directly from your mobile:

| Action | Description |
|--------|-------------|
| AI Chat | Send messages and receive streaming AI responses |
| Shell Commands | Execute terminal commands (with approval) |
| Browser Control | Navigate, reload, close tabs |

### QR Code Pairing

- **Faster Scanning**: Improved QR detection
- **Auto-Connect**: Remembers paired devices
- **Status Display**: Shows connection quality

### Sync Settings

New SyncSettingsPage with:
- Connection status indicator
- Paired devices list
- Auto-connect toggle
- Sync preferences

---

## Bug Fixes

1. **QR Scanner**
   - Improved camera handling on iOS
   - Better error messages for failed scans

2. **TypeScript**
   - Fixed TypeScript types for WiFiSyncService
   - Fixed process.defaultAgent type

3. **Desktop Control**
   - Fixed clipboard sync loop prevention
   - Added proper echo prevention
