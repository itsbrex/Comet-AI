# ☄️ Comet AI — v0.2.1 "The Control Update"
> *A massive leap in agentic reliability and visual intelligence.*

Welcome to **Comet v0.2.1**. This update, nicknamed "**The Control Update**," focuses on evolving Comet from a simple AI assistant into a robust **Agent Runtime Engine**. We've rebuilt the command orchestration from the ground up and restored the browser's most powerful capabilities.

---

## 🚀 Key Highlights

### 🧠 Neural Synthesis Engine (v2)
We've replaced the fragmented "action-then-response" loop with a unified **Synthesis Engine**. 
- **The Flow**: User Input → Agent Action Execution → Result Harvesting → **Final Synthesis**.
- **The Result**: The AI "sees" the results of its web searches and shell commands *before* it speaks to you, leading to 100% more coherent and factual answers.

### 🛡️ Intelligence Command Classification
To prevent "explanation loops" and orchestrator hangs, we've introduced **Command Scoping**:
- **Executable Commands**: Real-world operations like `[NAVIGATE]`, `[SHELL_COMMAND]`, and `[GENERATE_PDF]`.
- **Meta Commands**: Informational tags like `[EXPLAIN_CAPABILITIES]` and `[LIST_OPEN_TABS]`.
- **Context-Aware Parsing**: The AI now ignores commands trapped inside **markdown tables** or **code blocks**, allowing it to teach you how to use a tag without accidentally executing it!

### ✨ Visual Intelligence Registry
Confused about what Comet can do? Simply click the new **Sparkles icon** in the header or type `[EXPLAIN_CAPABILITIES]` in the chat.
- **Strict List**: A beautiful, glassmorphic panel listing every available tool, its description, and a real-world example.
- **Instant Discovery**: Discover powerful tags like `[WEB_SEARCH]`, `[SCREENSHOT_AND_ANALYZE]`, and `[SET_THEME]`.

---

## 🛠️ Restored & Restructured Features

### 🌐 Advanced Web Search (RAG)
`[WEB_SEARCH]` has been upgraded with **Neural Retrieval (RAG)**. 
- It doesn't just search; it **navigates the browser** so you can see the results while it extracts the relevant data into its memory.

### 📄 Professional PDF Generator
Restored `[GENERATE_PDF]` with a fixed parameter engine. 
- Generates branded, clean HTML-to-PDF documents featuring the **Comet AI "The Control Update"** watermark.

### 💻 Deep System Control
- **Shell Access**: Restored `[SHELL_COMMAND]` with a high-risk user permission layer.
- **Local Power**: Native support for **GPT-OSS-Cloud 120B** models directly via Ollama. 
- **Hardware Warnings**: New indicators to let you know if a model (like a 70B+) is too heavy for your current RAM.
